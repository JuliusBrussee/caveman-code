title: Quick Start
description: Get started with CAVE CLI in 5 minutes
---

## Start a session

```bash
cave
```

Type a prompt and the agent will help you code. Use `/help` for available commands.

## Configure a provider

```bash
export ANTHROPIC_API_KEY=sk-...
cave
```

Supports: Anthropic, OpenAI, Google, AWS Bedrock, Mistral, and more.
