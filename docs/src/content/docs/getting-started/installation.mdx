---
title: Installation
description: Install the CAVE CLI coding agent
---

## npm (recommended)

```bash
npm install -g cave
```

## Homebrew

```bash
brew install cave-cli/tap/cave
```

## curl

```bash
curl -fsSL https://cave.sh/install | sh
```

## Docker

```bash
docker run --rm -it ghcr.io/juliusbrussee/caveman-cli:latest
```

## Verify

```bash
cave --version
```
